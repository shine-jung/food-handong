## 🖖 Introduction

[한동맛나](https://food.handong.us/)는 한동인들을 위한 맛집 웹 어플리케이션입니다.

> 양덕, 영일대 등 한동대 근처의 100개가 넘는 맛집을 확인 할 수 있습니다.  
> 문의 및 피드백, 식당 추가 요청은 사이트 상단의 피드백 버튼을 이용해 주세요!

## 🧩 Main Functionalities

> - 🔍 식당 검색
> - 🥘 식당 상세정보 보기
> - ❤️ 식당 즐겨찾기
> - ⭐️ 평점과 리뷰 남기기
